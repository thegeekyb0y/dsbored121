# Kraked: Study Together

**Kraked** is a real-time collaborative productivity platform designed to help users focus and study together. It features synchronized global timers, task management, and detailed progress tracking.


[**Check it out Live**](https://dsbored121.vercel.app/)

- **Real-Time Study Rooms:** Live presence, shared sessions, private rooms, host controls

- **Task Management:** Priority levels, due dates, smart sorting, quick actions

- **Profile & Customization:** Avatars, custom subjects, status, study history
